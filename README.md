# 월간 CMDS 17회차 — 박준 파트 공유 자료

> **주제: "원하는 정보를 AI가 알아서 찾아오게 하기"**
> 공식 API가 없어도, 내가 원하는 정보를 AI가 알아서 가져오게 만드는 **정보 수집 4단 사다리** + 그걸 LLM이 대화 중 스스로 호출하게 **MCP/하네스로 패키징**하는 법, 그리고 수집한 자료를 검증된 지식으로 쌓는 **LLM Wiki 캡스톤**까지.

이 repo는 CMDSPACE 운영진(요한·태극) 공유용으로, 박준 파트의 **강의자료 + 실습키트 + 슬라이드 링크**를 담았습니다.

---

## 📅 17회차 개요

- **강의일**: 2026-06-01 (월)
- **시리즈 위치**: 정보 인입(인박스) 편 — 다음 회차는 "맥락·명령 주입" 편으로 확장
- **핵심 메시지 3개**
  1. **위에서부터 내려와라** — 크롤링부터 짜지 말고 "공식 API가 있나?"를 먼저 물어라.
  2. **바퀴를 다시 만들지 마라** — 이미 패키징된 MCP/Skill(arXiv·법제처·Firecrawl)을 찾아 붙여라.
  3. **외부 콘텐츠 = 잠재적 공격면** — 수집한 데이터는 "명령"이 아니라 "데이터"로 취급(프롬프트 인젝션 방지).

## 🏁 실습 시작법 (3스텝 요약)

수강생은 zip만 받으면 됩니다. 하네스(CLAUDE.md·슬래시커맨드·권한)가 **이미 다 세팅된 상태**입니다.

```
① 월간CMDS_17회차_실습키트.zip 압축을 풀고
② 폴더(cmds17-research-lab)에서 터미널을 열어 claude 실행
③ Claude에게  /start  라고 입력
```

`/start`가 현재 진행 상황을 파악하고 다음 할 일을 안내합니다. 막히면 언제든 다시 `/start`.
API 키 발급 절차는 키트 내 `api_keys/00_api_key_guide.md`에 단계별로 정리(전부 무료 티어로 실습 가능).
